BLACKJACK PROJECT - Dylan van Moorsel - TIA4v2a

---------------------------------------------------------------------------------------------

Binnen dit project vind je de gegevens van mijn Blackjack programma.

Blackjack is een simpel spel waarin 1 dealer en 1 speler tegen elkaar spelen. Je speelt zelf als de dealer waardoor je de opties krijgt om te : shuffelen, het spel te starten en de kaarten uit te delen.

---------------------------------------------------------------------------------------------

In deze github branch zijn de volgende functies beschikbaar om te testen :

Shuffle (Schud het deck voordat het spel kan worden gespeeld)

Start game (Hiermee start je de game na het schudden)

Deal (Deel de kaarten uit nadat je de vorige twee acties hebt gedaan)

Hit (Trek een extra kaart als dealer wanneer je onder de 17 punten zit)

Stand (Stop met trekken en bepaal de winnaar)

Feedback (Na elke keuze krijg je feedback of je correct hebt gespeeld)

Punten systeem (Correcte keuzes leveren 1 punt op, foute keuzes kosten 1 punt. Bij 2 strafpunten wordt het spel gestopt)

---------------------------------------------------------------------------------------------

Vereisten :

Visual Studio 2022 of hoger

.NET 10.0

---------------------------------------------------------------------------------------------

Installatie :

1. Clone het project via GitHub

2. Open Blackjack.sln in Visual Studio

3. Zorg dat de Cards map in de bin/Debug map staat

4. Run het project
